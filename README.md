# 📧 AI Email Generator

An AI-powered Email Generator built using **LangChain**, **Groq LLM**, and **Streamlit**. It generates professional emails for different scenarios with customizable tone and length.

---

## 🚀 Features

- Generate professional emails using AI
- Multiple email types
  - Leave Request
  - Job Application
  - Resignation
  - Complaint
  - Meeting Request
  - Business Proposal
  - Thank You
- Adjustable email tone
- Adjustable email length
- Dynamic input forms
- Email Preview & Edit Mode
- Download as TXT
- Download as DOCX
- Responsive Streamlit UI
- Modular project architecture

## ⚙️ Installation

Clone the repository

```bash
git clone <your_repository_url>
```

Go inside the project

```bash
cd AI_Email_Generator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
GROQ_API_KEY=your_api_key
```

Run the application

```bash
streamlit run app.py
```
